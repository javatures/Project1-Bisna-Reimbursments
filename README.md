**Project 1 - Reimbursments**

A web application for managing reimbursement ticket submissions and approvals between employees and their managers.


### Tools & APIs
- [x] Agile User Stories
- [x] Java SE 8
- [x] Gradle
- [x] JDBC
- [x] PostgreSQL
- [x] JavaEE Servlet
- [x] HTML/JS/CSS
- [x] AJAX/Fetch

**Getting Started**


Download the files and do gradle run
